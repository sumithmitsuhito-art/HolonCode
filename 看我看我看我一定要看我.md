## 部署方法

把整个 `DSPark-Code` 文件夹复制到目标电脑，在终端运行：

```
cd 你复制的路径
python boot.py
python -m atri.main
```

`boot.py` 会自动完成：

1. `pip install -e .` — 安装 atri + 核心依赖（httpx, rich）
2. 检测 `aiotieba` 是否已安装 → 未安装则询问是否自动安装
3. 配置 DeepSeek API Key
4. 配置贴吧登录凭据（可选，不影响浏览功能）

**唯一要求：目标电脑有 Python 3.12+ 和网络。**`tieba` 文件夹不需要带，aiotieba 从 PyPI 自动下载。不需要的贴吧依赖也可以跳过。

## 功能

- **Function Calling 工具系统** — 文件读写、搜索、记忆管理、贴吧浏览，AI 自动调用工具完成任务
- **贴吧浏览** — 浏览任意贴吧帖子、搜索话题、查看用户信息（新增）
- **技能系统** — 可插拔的行为模块，AI 自主激活/关闭/创建技能，支持渐进式加载
- **联网搜索** — 通过 Bing 获取实时信息
- **用户画像** — 自动记录用户偏好和习惯
- **上下文压缩** — 长对话自动总结，不会丢失上下文
- **可定制提示词** — 通过 `data/SOUL.json`、`RULES.json`、`CAPABILITY.json` 自定义角色和行为

## 技能系统

技能是 `skills/<技能名>/SKILL.md` 文件，AI 可以自主调用工具来激活、关闭、创建和修改技能。

最多同时激活 3 个技能，超出时自动关掉最早的。

内置技能：

- `code-reviewer` — 傲娇编程导师
- `translator` — 中/英/日三语翻译
- `storyteller` — 交互式故事创作
- `confidant` — 深度倾听模式

## 常见问题

**Q: 启动报 "api配置错误"** — 运行 `python boot.py` 重新走一遍配置，或单独运行 `atri-setup` 只改 API 配置。

**Q: 贴吧工具不可用** — 运行 `python boot.py` 选择安装 aiotieba，或手动 `pip install aiotieba`。

**Q: 贴吧帖子看不到内容** — Windows 上 aiohttp 的 TLS 握手可能超时，ati 启动时已自动修复（HTTPS→HTTP）。如仍有问题，尝试配置 BDUSS 和 STOKEN（运行 `python boot.py` 或 `atri-setup`，在第四步从浏览器 cookie 获取）。

**Q: 网络请求失败** — 检查网络和 API URL 是否正确。

**Q: 如何重置对话** — 删除 `data/ConversationHistory.json`。

**Q: 如何修改角色** — 编辑 `data/SOUL.json` 后重启。
